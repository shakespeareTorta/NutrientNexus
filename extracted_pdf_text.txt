PoC: sensor stubbing

Technical limitation
Cannot add real agricultural sensors to Turtlebot.

PoC solution
     Stub sensor getter functions to return controlled values per crop zone.
     Simulate fertiliser application by rotating the TurtleBot in place at
     the selected zone.




  Team 38: shakespeareTorta       NutrientNexus             May 19, 2026     24 / 26
PoC: sensor stubbing

Technical limitation
Cannot add real agricultural sensors to Turtlebot.

PoC solution
     Stub sensor getter functions to return controlled values per crop zone.
     Simulate fertiliser application by rotating the TurtleBot in place at
     the selected zone.

Why this is still valid
The PoC tests the key aspect of technical solution — environment-based
agricultural decision-making system.

                    scan → weigh in data → decide → act/report
